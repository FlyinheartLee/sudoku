# 灵感胶囊 - 设计规格文档

## 项目概述

一个纯本地的灵感记录 App，支持语音/文字输入，AI 自动整理归类，支持后续 AI 讨论。

## 技术栈

- **框架**: Flutter 3.x
- **语音识别**: 科大讯飞语音听写
- **AI 处理**: 腾讯混元 / Kimi / 其他 OpenAI 兼容 API
- **本地存储**: SQLite + SharedPreferences
- **状态管理**: Provider / Riverpod

## 核心功能

### 1. 输入模块
- 语音输入：长按录音，调用科大讯飞 SDK
- 文字输入：多行文本框
- 混合模式：可以先语音，再补充文字

### 2. AI 整理模块
- 发送原始文本到 AI
- 返回整理后的 Markdown
- 提取标签和分类建议
- 用户确认/编辑/放弃

### 3. 分类系统
- 默认分类：灵感、感悟、待办、读书、随笔
- 自定义分类
- AI 建议分类（可修改）

### 4. 讨论模块
- 点击记录进入详情
- "和 AI 讨论"按钮
- 讨论记录追加到原文下方
- 支持连续多轮讨论

### 5. 存储模块
- SQLite 存储记录
- 本地 Markdown 文件备份
- 支持导出

## API 配置

### 科大讯飞
- 需要申请 AppID、APIKey、APISecret
- 使用语音听写 API

### AI 模型
- 支持 OpenAI 兼容格式
- 默认使用腾讯混元 API
- 可配置其他国内模型（Kimi、文心等）

## 数据模型

class Inspiration {
