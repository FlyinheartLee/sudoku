# 2026-03-20 记忆

## 益脑游戏库 Brain Games

**GitHub 仓库**: https://github.com/FlyinheartLee/sudoku
**项目性质**: H5 益智游戏合集
**技术栈**: 纯 HTML/CSS/JavaScript
**部署方式**: GitHub Pages
**主页面**: https://flyinheartlee.github.io/sudoku/index.html

### 游戏列表（10款）

| 序号 | 游戏 | 文件 | 描述 |
|------|------|------|------|
| 1 | 🧩 数独 | sudoku-h5.html | 经典数字逻辑，4种难度 |
| 2 | 🔢 数字迷宫 | number-maze.html | 从1走到N，按顺序走相邻格子 |
| 3 | 🏢 摩天楼 | skyscraper.html | 边缘数字表示可见建筑数 |
| 4 | 🔢 2048 | game2048.html | 滑动合并数字 |
| 5 | 🧩 华容道 | klotski.html | 滑块拼图，复原数字顺序 |
| 6 | 💣 扫雷 | minesweeper.html | 经典扫雷游戏 |
| 7 | 🔧 接水管 | pipe.html | 旋转管道连通水源到目标 |
| 8 | 🃏 记忆翻牌 | memory.html | 翻牌配对游戏 |
| 9 | 📦 推箱子 | sokoban.html | 经典推箱子到目标位置 |
| 10 | 🌉 数桥 | hashi.html | 连接岛屿，数字表示桥的数量 |

### 统一特性
- 每款游戏顶部都有 🎮 切换按钮
- 点击展开菜单可跳转到其他游戏
- 统一配色风格，各游戏有自己的主题色
- 支持移动端触摸操作

### 开发教训
- 文件编辑后要检查完整性（</html>数量）
- 部署前用 curl 验证线上文件
- GitHub Pages 有缓存，需要强制刷新
